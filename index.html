    CREDIT RISK PROJECT

    This project focused on analyzing and predicting customer credit risk to help businesses make data-driven decisions. I cleaned and prepared the dataset by handling missing values, duplicates, and inconsistent entries, then performed exploratory data analysis (EDA) to uncover key patterns such as default trends, income distribution, and payment history. Using statistical techniques and visualization dashboards, I identified high-risk customer segments and provided actionable insights to minimize loan defaults. This solution reduced manual reporting time and improved risk assessment accuracy, demonstrating how data analytics can support financial institutions in balancing growth with risk management.

    CUSTOMER CHURN PROJECT

    This project aimed at understanding and predicting customer churn to help businesses improve retention strategies. I began with thorough data cleaning and preparation, addressing duplicates, missing values, and inconsistent entries to ensure data accuracy. Through exploratory data analysis (EDA), I uncovered patterns in customer behavior, such as contract type, tenure, payment method, and service usage, that strongly influenced churn. I then built an interactive dashboard to visualize churn trends, highlight at-risk customer segments, and track key performance indicators. The analysis provided actionable insights that businesses can leverage to reduce churn rates, strengthen customer loyalty, and optimize marketing campaigns.

    AMAZON CUSTOMER REVIEW PROJECT

    This project explores an Amazon e-commerce dataset containing product details, pricing information, discounts, ratings, and customer reviews. I performed extensive data cleaning and transformation, fixing text encoding issues, standardizing brand names, and preparing the data for analysis. The analysis focused on understanding brand performance, pricing strategies, and customer behavior. I extracted brand-level insights, compared actual vs discounted prices, and evaluated how discount percentages relate to customer engagement through ratings and reviews. Key insights highlighted which brands dominate their categories, how discounts affect buyer trust, and which product types receive the most positive feedback. This project demonstrates how data-driven insights can support better pricing decisions, targeted marketing, and improved customer experience in the e-commerce space.
